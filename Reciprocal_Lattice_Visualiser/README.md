# 🌐 3D Reciprocal Lattice Visualizer

An interactive **3D web application** built with **Streamlit** to visualize crystal lattices in both **real space** and **reciprocal space**.  
Explore fundamental concepts of solid-state physics, including **Wigner-Seitz cells** and **Brillouin zones**, with intuitive controls.

## ✨ Features

### ⚛️ Multiple Lattice Types
Visualize common crystal structures like:
- Simple Cubic (SC)
- Body-Centered Cubic (BCC)
- Face-Centered Cubic (FCC)

### 🔬 Interactive 3D Plot
Rotate, pan, and zoom the lattice structures for a complete 360° view.

### 🎛️ Real-Time Controls
Instantly adjust parameters such as:
- Lattice constant (**a**)
- Number of unit cells (**N**)  
to see how they affect the structures.

### 👀 Visibility Toggles
Independently show or hide:
- 🔵 Real-space Lattice  
- 🔴 Reciprocal-space Lattice  
- 🟦 Wigner-Seitz Cell  
- 🟥 First Brillouin Zone  

### 📸 Persistent Camera
The camera's position is saved, so your viewing angle remains consistent as you adjust the lattice parameters.
